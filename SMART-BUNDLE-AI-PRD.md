# Project: SmartBundle AI - Shopify App
**Motto:** Boost Average Order Value (AOV) via AI-powered Bundles.(Subscription-based).

---

## 1. Technical Stack (Latest 2026 Standards)
- **Framework:** React Router v7 (Remix successor) using Shopify App Template.
- **UI Architecture:** Shopify Polaris Web Components (Preferred over React Polaris for storefront and modern admin feel).
- **Backend:** Node.js with Vite.
- **Database:** Prisma with SQLite (Local) / PostgreSQL (Production).
- **Discount Logic:** Shopify Functions (Product Discount API) - *Mandatory for Checkout integration.*
- **AI Engine:** Simple Association Rule Mining (backend-based) for product suggestions.

---

## 2. Core Features & Requirements

### A. Merchant Admin Features
1. **Analytics Dashboard (App Home):**
   - Display total revenue generated by bundles.
   - Active vs. Inactive bundles list.
   - Use `<ui-layout>`, `<ui-card>`, and `<ui-text-block>` web components.

2. **Bundle Creator (Form):**
   - **Fields:** Title, Discount Type (Fixed/Percentage), Discount Value.
   - **Product Picker:** Use App Bridge v4 `shopify.resourcePicker` to select products.
   - **Quantity Breaks:** Allow "Buy 2 Get 10%, Buy 3 Get 20%" logic.

3. **Status Management:**
   - Toggle bundles ON/OFF instantly.

### B. Customer Facing (Storefront) Features
1. **Dynamic Bundle Widget:**
   - Use **Theme App Extensions (App Blocks)**.
   - Modern "Bundle Card" UI with product images and variant selectors.
   - Real-time price updates when variants change.

2. **Auto-Discount:**
   - No coupon codes. Discounts must apply automatically via Shopify Functions when bundle criteria are met.

## 3. Database Schema (Prisma)
```prisma
model Bundle {
  id             String   @id @default(uuid())
  shop           String
  title          String
  status         String   @default("active") // active, inactive
  discountType   String   // percentage, fixed_amount
  discountValue  Float
  products       Json     // Store array of product IDs and variant IDs
  createdAt      DateTime @default(now())
  updatedAt      DateTime @updatedAt
}

model Analytics {
  id             String   @id @default(uuid())
  bundleId       String
  revenue        Float
  orderId        String
  createdAt      DateTime @default(now())
